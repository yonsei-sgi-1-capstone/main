# 캡스톤 프로젝트

## Course Description

이번 과정에서는 머신러닝을 활용하여, 실제 프로젝트를 수행하는 과정에 대해서 배웁니다.



## Prerequisites:  
파이썬 기본

## Learning Outcomes
과정을 완료하고 나면, 다양한 문제에 대해서 머신러닝을 적용하여 계획, 데이터 수집, 훈련, 평가하는 방법을 익힐 수 있습니다.
